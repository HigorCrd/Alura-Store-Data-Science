# Alura-Store-Data-Science

## Sobre o projeto

Projeto do desafio de Data Science da Alura. A ideia é analisar os dados de 4 lojas do Senhor João pra ajudar ele a decidir qual loja faz mais sentido vender.

Foram usados dados de vendas em CSV disponibilizados no repositório do desafio.

## O que foi analisado

- Faturamento total de cada loja
- Categorias de produtos mais vendidas
- Avaliação média dos clientes (nota de 1 a 5)
- Produtos com mais e menos vendas
- Custo médio do frete

## Ferramentas utilizadas

- Python 3
- Pandas
- Matplotlib
- Google Colab


## Principais resultados

| Indicador | Destaque |
|-----------|----------|
| Menor faturamento | Loja 4 (R$ 1.384.497,58) |
| Melhor avaliação | Loja 3 (4.05) |
| Pior avaliação | Loja 1 (3.98) |
| Frete mais barato | Loja 4 (R$ 31.28) |

A Loja 4 tem o frete mais baixo e uma avaliação ok, mas mesmo assim fatura menos que todas as outras.

## Conclusão

A loja que deve ser vendida é a **Loja 4**. Apesar do frete barato e avaliação razoável, o faturamento dela é o menor entre as quatro, o que mostra que ela não está performando bem em vendas comparado com as outras unidades.
